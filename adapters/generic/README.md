# Agent-Cite — Generic Prompts

For LLMs without native plugin support, use the prompts in `prompts/` directly:

1. **To audit evidence**: Paste `prompts/cite-audit.md` into your conversation
2. **To fix violations**: Paste `prompts/cite-fix.md`
3. **To get a project report**: Paste `prompts/cite-report.md`

These are the LLM-agnostic core prompts that all platform-specific adapters wrap.
